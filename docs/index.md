# IML Football Challenge Documentation

Welcome to the official documentation for the **IML Football Challenge** application. 

This project is a modern, responsive web application for managing football predictions and tournament tracking. It is built using the latest web technologies to ensure a scalable and performant user experience.

## 🚀 Key Technologies

- **Frontend Framework**: [Angular v22](https://angular.dev/)
- **Styling & Design System**: [Tailwind CSS v3.4](https://tailwindcss.com/)
- **Backend/SSR Engine**: [Node.js](https://nodejs.org/) & [Express v4](https://expressjs.com/)
- **Testing**: [Vitest](https://vitest.dev/) (Unit) & [Playwright](https://playwright.dev/) (E2E)

## 📖 How to use this documentation

Use the navigation bar (or the hamburger menu on mobile) to explore the various aspects of the system:

- **[Architecture](architecture.md)**: Learn about the core design patterns (Smart/Dummy components, Facades, Utils) that structure the application.
- **[Components](components.md)**: Explore the custom Tailwind CSS design system used across shared UI components.
- **[Setup & Deployment](setup.md)**: Find the exact commands needed to run the app locally, develop with Server-Side Rendering (SSR), and build for production.
- **[Testing](testing.md)**: Learn how to execute and write tests using Vitest and Playwright.
